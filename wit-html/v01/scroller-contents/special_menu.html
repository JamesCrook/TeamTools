The Audacity "What is That?" web page shows the main components of the Audacity User Interface. You can find out more about these components.

The menus work too.

The 'Reset' button gets you back to the starting screen.
  • Enable annotation-capture mode. Background fades completely to white and only the arrows from the labels immediately around the object are shown. Useful for capturing annotated screenshots of toolbars.
