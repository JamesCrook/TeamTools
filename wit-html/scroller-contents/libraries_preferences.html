Libraries Preferences



Both the optional MP3 Export Library and the optional FFmpeg Import/Export Library allow Audacity to import and/or export many additional audio formats such as AC3, AMR(NB), M4A and WMA, and to import audio from video files.
Accessed by: Edit > Preferences > Libraries    (on a Mac Audacity > Preferences > Libraries )
Libraries Preferences.
Click on any other Preferences sections in the above image to read about those Preferences.

MP3 Export Library

  • MP3 Export Library Version: This shows the current version of the LAME MP3 encoding library being used by Audacity for MP3 export. If a suitable version of LAME is not accessible to Audacity, the message will instead state that the library is "not found". In that case, use the provided buttons to either locate LAME on your computer or download a free copy of it, following the instructions on the FAQ:Installation, Startup and Plugins page.
  • Locate If you already have the LAME library on disk and Audacity cannot find it, use this button to tell Audacity where it is.
  • Download Use this to fetch a copy of the LAME library.
