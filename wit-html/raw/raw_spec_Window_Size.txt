This is the spec for the interactive diagram which explains window size in spectrograms:

<pre>
DISPLAY_FOCUS_DETAIL Focus=SmallCircle
  BUTTON_ROW
  IMAGE_WITH_HOTSPOTS
</pre>



The specification above builds a display with labels shown in button style, stacked above a focus-detail display which uses a small circle as the focus. The exact details come from the rest of this page, and can be edited pretty much in the normal way.  There are seven buttons for selecting the window size, each of which loads one of the seven spectrogram images.  There is a false-colour hotspot image map (it looks a bit like a TV test card) that defines the hotspot zones, most of which are for explaining the different sounds in the spectrogram.

Swap the order of BUTTON_ROW and IMAGE_WITH_HOTSPOTS to have the buttons under the focus-detail panel instead.  Use a HORIZONTAL_LAYOUT to have the buttons on the left.  Use OVERLAY_LAYOUT to have the button painted over (on top of) the spectrogram image.  There's an [https://wit.audacityteam.org/imagemap2_test.htm?page=Window_Size interactive demo].

<pre>NEXTZONE:COMMENT=freq;TIP=</pre>
<h2>Frequency Scale</h2><em><p>The spectrogram is a plot showing how the frequencies of a sound vary over time. The <b>frequency scale</b> runs from 0 KHz to 8 KHz.</p><p>This spectrogram shows a plot for the word 'Audacity'. </p><p>The most intense sounds are shown in white, then red, purple, blue, with black indicating no intensity at that frequency at that time.</p></em>

<pre>NEXTZONE:COMMENT=win-size;TIP=</pre>
<h1>Window Size</h1><em><p>Small windows sizes measure frequency over a short time range.  This is good when the frequencies change quickly.</p><p>Where the frequencies are sustained over longer time periods, larger window sizes give more precision about what the frequencies are.</p><p>The number for 'window size' is 'the number of audio samples'.</p></em>

<li>Hold shift down to freeze the hotspot.
